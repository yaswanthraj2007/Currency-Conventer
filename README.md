💱 Currency Converter Web App

📌 Description:

This is a simple and modern Currency Converter web application that allows users to convert values between different currencies in real-time using an API.
The app fetches live exchange rates and provides accurate conversion results with a clean and responsive UI.

🚀 Features:

🌍 Convert between multiple currencies
🔄 Real-time exchange rates using API
⚡ Fast and responsive interface
🎨 Modern UI with animations and gradients
📊 Displays API status (Live / Loading / Error)
❗ Input validation for correct values

🛠️ Technologies Used:

HTML5
CSS3 (Advanced styling + animations)
JavaScript (Vanilla JS)
ExchangeRate API

📂 Project Structure:

├── index.html   # Main HTML structure
├── style.css    # Styling and UI design
├── script.js    # Logic and API handling

📸 How It Works:

->Enter the amount
->Select "From" currency
->Select "To" currency
->Click Convert
->Result will be displayed instantly
